# AI Income Trends

Daily research reports by Logan Knox tracking how Claude Code and AI tools are being used to generate passive and traditional income.

## Report Structure

Each daily report covers:
- **Established Trends** — Proven, viable AI income methods with saturation risk assessment
- **Cutting Edge & Speculative** — Emerging use cases before they go mainstream
- **Saturation Watch** — Fields becoming oversaturated to deprioritize
- **Top Recommendations** — 1-2 highest-leverage opportunities to act on now

## Reports

Reports are saved to `reports/YYYY-MM-DD.md`.
